### Assigning IP Variables
```bash - kali
export KALI=$(ifconfig tun0 | grep -w inet | awk '{print $2}' | tr -d " \t\n\r") && export TARGET=
```
```bash - kali
mkdir $TARGET && cd $TARGET
```
Now open you're other terminals and repeat creating variables.
