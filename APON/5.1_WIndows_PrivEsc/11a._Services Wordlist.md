This is an experiement to try and identify unusual services running on Window target.

```command prompt - windows
cd %TEMP% && sc query state= all | findstr "SERVICE_NAME:" >> Servicenames.txt
```

```command prompt - windows
FOR /F %i in (Servicenames.txt) DO echo %i type Servicenames.txt
```

```command prompt - windows
FOR /F "tokens=2 delims= " %i in (Servicenames.txt) DO @echo %i >> services.txt
```

```command prompt - windows
FOR /F %i in (services.txt) DO @sc qc %i | findstr "BINARY_PATH_NAME" >> path.txt
```


Now, open the services.txt file, copy over to kali, and run diff against the following wordllist:

```bash - kali
diff <(sort services.txt) <(sort servicesWordlist.txt) -y -Z
```
